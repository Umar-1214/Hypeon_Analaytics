Put your logo here to use it in the Copilot sidebar and main screen.

- Name the file: logo.png (or logo.svg)
- Or set VITE_APP_LOGO_URL in frontend/.env to your image path (e.g. /logo.svg)

If logo.png / logo.svg is present, it is used automatically. If the file is missing or fails to load, the default gradient logo is shown.
